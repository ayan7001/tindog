What is vanilla CSS?

Vanilla CSS is just CSS, neither “Less CSS” , nor any external CSS framework.

vanilla CSS vs Bootstrap

Vanilla CSS gives you complete freedom to design your page. The code you write for your page is very specific, hence your page loads up faster.
On the other hand, the Bootstrap framework may have a lot of unnecessary code which may cause your page to take longer than usual to load up. Bootstrap is an external CSS framework which may conflict with your own CSS code. You do not have absolute freedom to design your page. But, you can definitely build your page faster with Bootstrap than Vanilla CSS.
